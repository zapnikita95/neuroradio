Настоящее Соглашение регулирует отношения между сервисом «Эфир AI» (далее — «Сервис») и пользователем (далее — «вы»). Начиная использовать Сервис, вы подтверждаете согласие с условиями Соглашения, Офертой и Политикой конфиденциальности.

1. Общие положения

Сервис предоставляет программное обеспечение для Android и расширение для браузера, которое формирует и озвучивает короткие истории о музыке во время прослушивания.

  • Приложение Android использует доступ к уведомлениям только для чтения метаданных музыкальных плееров; прочие уведомления не обрабатываются.

Оплата производится через ЮKassa пошагово: выбор тарифа → указание email и согласие с документами → оплата на защищённой странице ЮKassa → активация доступа по email. Подробный порядок — в Оферте, раздел 4. Автопродление не подключается.
