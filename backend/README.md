# Music Story BFF

Standalone Express backend for APK testing: enriches track metadata, generates a short Russian fan-radio story, and optionally synthesizes speech via Yandex SpeechKit.

## Requirements

- Node.js 18+
- Optional: [Groq API key](https://console.groq.com) and [Yandex Cloud SpeechKit](https://cloud.yandex.com/docs/speechkit/) credentials

Without API keys the server runs in **demo mode** and returns a TTS-friendly Russian script for Android `TextToSpeech` (no server-side audio).

## Setup

```bash
cd backend
npm install
cp .env.example .env
# Edit .env and add keys (optional for demo mode)
npm run dev
```

Server starts at `http://localhost:3000`.

## Environment variables

| Variable | Description |
|----------|-------------|
| `GROQ_API_KEY` | Groq API key for story generation (`qwen/qwen3-32b`) |
| `YANDEX_API_KEY` | Yandex Cloud API key for SpeechKit |
| `YANDEX_FOLDER_ID` | Yandex Cloud folder ID |
| `PORT` | HTTP port (default `3000`) |

## API

### `GET /health`

Health check.

### `POST /v1/story/full`

Generate a full story response for a track.

**Request body:**

```json
{
  "artist": "Queen",
  "title": "Bohemian Rhapsody"
}
```

**Response (with API keys):**

```json
{
  "artist": "Queen",
  "title": "Bohemian Rhapsody",
  "year": 1975,
  "genre": "rock",
  "mbid": "...",
  "script": "вҖҰ",
  "word_count": 62,
  "voiceId": "filipp",
  "demo": false,
  "audioUrl": "/audio/<uuid>.ogg",
  "audioFile": "<uuid>.ogg",
  "sources": {
    "musicbrainz": true,
    "groq": true,
    "yandexTts": true
  }
}
```

**Demo mode** (`demo: true`): `audioUrl` is `null` and `ttsHint` explains to use Android TTS with the `script` field.

### `GET /audio/<filename>`

Static OGG Opus files generated by Yandex TTS.
