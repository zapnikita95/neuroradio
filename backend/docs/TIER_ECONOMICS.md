| **Free** (`:free` модели) | ~0 ₽ | ~2,1 ₽ | **~2,1 ₽** | **~100%** |

## 5. Сколько стоит пользователь в месяц (если выбивает лимит каждый день)

Формула: `дней × лимит/день × ₽/история` (берём **2,1 ₽** free, **2,3 ₽** paid).

| Сценарий | Историй / мес | Себестоимость / мес |
|----------|---------------|---------------------|
| Free **3/день** | 90 | **~189 ₽** |
| Free **5/день** | 150 | **~315 ₽** |
| Free **10/день** (сейчас) | 300 | **~630 ₽** |
| Trial 10/день | 300 | **~690 ₽** при 1 ₽ выручки |
| Premium 10/день (средний активный) | 300 | **~690 ₽** при 199 ₽ выручки |
| Premium 25/день (макс.) | 750 | **~1 725 ₽** |

### Точка безубыточности premium (199 ₽/мес)

При **~2,3 ₽/история**:

- **~86 историй / мес** ≈ **~2,9 истории / день** — ноль маржи (только себестоимость).
- Подписчик, который стабильно жмёт **10/день**, стоит **~690 ₽** → **убыток ~490 ₽/мес** на одного такого пользователя.
- Подписчик **25/день** на полной нагрузке → **убыток ~1 500 ₽/мес**.

**199 ₽ имеет смысл**, если средний premium реально генерирует **~3 истории / день**, а не 10–25.

## 6. Сколько платных нужно, чтобы «окупать» бесплатных

Упрощённо: один premium в среднем должен **перекрывать TTS free-пользователей** (LLM у free ≈ 0).

| Free-лимит | Себестоимость heavy free / мес | Premium subs на 1 heavy free* |
|------------|-------------------------------|-------------------------------|
| **3 / день** | ~189 ₽ | **~1,0** |
| **5 / день** | ~315 ₽ | **~1,6** |
| **10 / день** | ~630 ₽ | **~3,2** |

\* При допущении: каждый premium даёт **~199 ₽** и в среднем потребляет **~86 историй/мес** (своя точка безубыточности). Маржа с premium ≈ **199 − 86×2,3 ≈ 0 ₽** → один premium ≈ один «слот» ~189–630 ₽ free-TTS.

**Практика:** большинство free **не** бьёт лимит каждый день. Реальная цифра — **30–50% от max** → умножай таблицу на **0,3–0,5**.

Пример: free **10/день**, но среднее **3/день** → ~189 ₽/мес → **~1 premium на 1 активного free**.

## 7. Альтернативы Yandex для free (РФ, без Groq/Gemini/ElevenLabs)

| Вариант | ₽ / 1M символов | ₽ / история (~1 550 sym) | Работает из РФ | Комментарий |
|---------|-------------------|--------------------------|----------------|-------------|
| **Yandex SpeechKit v1** (сейчас) | **1 316** | **~2,1** | да | базовый голос, уже в проде |
| **SaluteSpeech Freemium** (физлицо) | 0 до лимита | 0 | да | **200 000 sym / мес на один ключ** → ~**130 озвучек/мес на ВСЁ приложение**, не на пользователя |
| **SaluteSpeech пакет** (физлицо) | **1 000** | **~1,55** | да | чуть дешевле Yandex; нужен отдельный ключ |
| **SaluteSpeech юрлицо** | **~186** | **~0,29** | да | нужен ИП/ООО; **в 7× дешевле Yandex** — целевой масштаб |
| **Только текст без озвучки** | 0 | 0 | да | ломает продукт (радио без голоса) |
| **Android TTS на телефоне** | 0 | 0 | да | **не используем** по архитектуре (AGENTS.md) |
| Groq / Gemini / ElevenLabs / Azure | — | — | **нет / нестабильно** | для РФ не опора |

### Рекомендации по free-tier (экономика + продукт)

1. **Снизить free до 3–5 историй / день** — главный рычаг (10/день × 2,1 ₽ = **21 ₽/день** на heavy user).
2. **Free = короткая история 30 с** (~900–1 000 sym после markup) → **~1,2–1,4 ₽/история** вместо 2,1 ₽.
3. **Premium = 60 с + DeepSeek + опционально Salute premium-голос** (уже в `tts-router`).
4. **Общий пул Salute Freemium 200k** — только на раннем этапе с малым MAU; при росте быстро кончится.
5. **Долгосрочно:** оформить **юрлицо + SaluteSpeech** (~0,29 ₽/озвучка) или договор с Yandex — иначе при 10 free/день экономика не сходится.

Комбо «разумный free» (пример):

| Параметр | Значение | Себестоимость heavy user / мес |
|----------|----------|----------------------------------|
| 3 истории / день, 30 с, Yandex | 90 × ~1,3 ₽ | **~117 ₽** |
| 5 / день, 30 с | 150 × ~1,3 ₽ | **~195 ₽** |
| 3 / день, 60 с | 90 × ~2,1 ₽ | **~189 ₽** |

## 8. Trial 1 ₽/мес

Воронка, не прибыль: **~690 ₽** себестоимости при 10/день vs **1 ₽** выручки.  
Имеет смысл как **конверсия в 199 ₽**, с жёстким дневным потолком (10 ок, если мало подписчиков trial).

## 9. Premium 199 ₽/мес — что закладывать в модель

| Метрика | Значение |
|---------|----------|
| LLM (DeepSeek, 750 hist max) | ~**$1,28** ≈ **128 ₽** (старый расчёт; при полной нагрузке) |
| TTS (750 × 2,1 ₽) | **~1 575 ₽** |
| **Итого при 25/день каждый день** | **~1 700 ₽** vs 199 ₽ |

**Рабочая модель ценообразования:** 199 ₽ покрывает **~90 озвучек / мес** (~3/день). Лимит **25/день** — антиабьюз burst, не «ожидаемое потребление».

## 10. Шпаргалка (чтобы не потерять)

```
Yandex TTS:     1 316 ₽ / 1M sym  →  ~2,1 ₽ / история (60 с, с markup)
DeepSeek LLM:   ~0,17 ₽ / история
Free LLM:       0 ₽
Полная история: ~2,1 ₽ (free)  |  ~2,3 ₽ (paid)

Free 10/день max:   ~630 ₽/мес TTS
Free 3/день max:    ~189 ₽/мес TTS
Premium breakeven:  ~3 hist / день
1 heavy free (10/d): нужно ~3 premium (если оба на max)
```
