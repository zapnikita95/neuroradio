# App Store — скриншоты iPhone

Папка **`screenshots/`** — 6 кадров **1290×2796** (6.7" Display).

| # | Сцена |
|---|--------|
| 01 | Нейроведущий + плеер |
| 02 | История в эфире |
| 03 | Архив рассказов и оценки |
| 04 | Дневник прослушиваний |
| 05 | Амплуа рассказчика |
| 06 | Выбор голоса |

Пересборка (исходники `скрин1.jpg` … на рабочем столе):

```bash
node scripts/generate-store-screenshots.mjs
```

В App Store Connect: **6.7" Display** или **6.5"** — загрузите `*-ios.jpg`.
