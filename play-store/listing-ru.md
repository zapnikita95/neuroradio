## Полное описание (до 4000 символов)

```
Эфир AI — нейросетевой радиоведущий для Android. Пока вы слушаете музыку в Яндекс Музыке, Spotify и других плеерах, приложение узнаёт текущий трек, находит интересные факты в открытых источниках и рассказывает короткую историю живым голосом между песнями.

Как это работает
• Приложение видит, что сейчас играет, через системные уведомления медиа (MediaSession).
• Сервис собирает факты о треке и артисте, ранжирует их по «интересности».
• Нейросеть пишет связный текст на русском в выбранном стиле.
• Голос Yandex SpeechKit озвучивает историю — как настоящий эфир.

6 амплуа ведущих
Радиоведущий, ночной диджей, эксперт, современный рассказчик, фанат, закулисье — у каждого свой характер, темп и подача.

Для кого
• Любителям музыки, которым интересен контекст вокруг треков.
• Слушателям в машине, на прогулке и дома — руки свободны, истории идут сами.
• Тем, кто хочет «умное радио» поверх своего плейлиста, без готовых заготовок.

Тарифы
• Базовый — ограниченное число историй в день.
• Расширенный — больше эфиров, приоритетная озвучка, подписка на сайте efir-ai.ru.

Важно
• Нужен доступ к уведомлениям медиа (для определения трека).
• Для озвучки и генерации используется сервер Эфир AI — нужен интернет.
• Приложение не заменяет музыкальный плеер: оно работает поверх уже установленных приложений.

Поддержка: efir-ai@yandex.ru
Сайт: https://www.efir-ai.ru
```

## Ключ подписи в Play Console

1. При **первой** загрузке AAB выбери: **«Разрешите Google защищать ключ подписи приложения»** (рекомендуется).
2. **Не** выбирай «Экспортировать ключ из Java Keystore» — это для миграции старых приложений.
3. AAB подписывается **upload-ключом** из `android/app/keystore/upload.keystore` (см. `android/PLAY_SIGNING.local.md`).
4. После загрузки: **Настройка → Целостность приложения → Подписание приложений** — там видны отпечатки upload и app signing.

## Загрузка AAB

Файл после сборки: **`efir-ai.aab`** в корне репозитория.

Внутреннее тестирование → Создать выпуск → Загрузить `efir-ai.aab` → Проверить versionCode → Опубликовать.
